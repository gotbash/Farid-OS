# Daily Pulse Report

Turn the supplied daily metrics into a short operating update.

Required output:

- Executive summary
- KPI movement vs yesterday and vs target
- What changed
- Top risks
- Top 3 priorities today
- Escalations / decisions needed

Rules:

- Use only supplied metrics and context.
- Separate observed movement from interpretation.
- Do not invent targets, budgets, inventory levels, or owners.
- Mark missing inputs explicitly with `[MISSING]`.
