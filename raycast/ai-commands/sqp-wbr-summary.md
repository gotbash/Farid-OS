# SQP WBR Summary

Turn the supplied Amazon Search Query Performance export into a concise Weekly Business Review summary.

Required output:

- Executive summary
- KPI snapshot
- Top query concentration
- Main funnel leak
- Strong query clusters
- Risks and gaps
- Recommended actions
- Decision needed
- Notion-ready summary

Rules:

- Use only evidence from the supplied SQP export.
- Separate observations from interpretation.
- Do not invent TACoS, spend, targets, or paid-search context if those are not present.
- Call out missing inputs explicitly with `[MISSING]`.
- Keep the output actionable and brief enough to paste into Notion.
