# Amazon Decision Engine

Given the supplied metric movement:

1. Restate only observed facts.
2. Build a ranked cause tree covering traffic, ad auction, conversion, retail readiness, price/promo, inventory, competition, seasonality, and measurement.
3. Specify the data check that would confirm or reject each cause.
4. Recommend prioritized actions with expected impact, reversibility, risk, owner, and measurement window.
5. Identify actions that should wait for more evidence.
