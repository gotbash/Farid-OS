# Raycast ↔ Notion setup

1. Install and authorize the official Notion Raycast extension if you want native workspace search.
2. Import the repository Quicklinks and snippets.
3. Use `vault` to open the canonical Knowledge Vault.
4. Use `ns` to open Notion search with a query.
5. Copy AI output into the appropriate Notion database after reviewing claims and owners.

The v1.2 scripts do not write to Notion automatically. This keeps Notion as the reviewed source of truth and avoids storing integration secrets.
